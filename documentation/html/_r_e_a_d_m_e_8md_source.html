11  Descrizione:
12  Realizzare un compressore/decompressore basato sull'algoritmo LZ78.
13 
14  Compilazione dell'applicazione:
15  Per la compilazione è presente un makefile quindi è sufficiente eseguire il comando "make".
16  La lista degli obiettivi presenti nel makefile risulta:
17  make [all]: Build the executable
18  make archive: Archive the executable, the makefile and all *.c and *.h files
19  make clean: Remove the excutable, *.lz78 and *.o files
20  make help: Show this info
21 
22  Esecuzione dell'applicazione:
23  Per lanciare l'applicazione è necessario eseguire l'eseguibile "lz78" utilizzando le seguenti opzioni:
24  -c --compress
25  Compress specified file
26  -d -u --decompress --uncompress
27  Uncompress specified file
28  -o -n --output --name
29  Set the name for the output file
30  -h --help
31  Show the help
32  -v --verbose
33  Display more messages
34  -f --force
35  Silent overwrite if the output file already exists
36  -e --extension
37  Use the input file extension for the output file
38  -p --path
39  Use the input file path for the output file
